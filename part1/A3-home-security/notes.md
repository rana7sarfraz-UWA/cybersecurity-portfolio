# A3 – Home Security

Several security measures are used at home.

1. WiFi Password Protection  
The home router requires a password to connect which prevents unauthorized access.

2. Router Firewall  
The router includes a firewall that blocks suspicious incoming network traffic.

3. Door Locks  
Physical locks prevent unauthorized entry into the house.

4. Antivirus Software  
Computers use antivirus software to detect malware and protect personal data.

These measures help protect both physical safety and digital devices.
