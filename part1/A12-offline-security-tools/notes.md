# A12 – Offline Security Tools

Offline security tools help protect systems without requiring internet connectivity.

Five examples include:

1. Wireshark  
A network analysis tool used to capture and inspect network packets.

2. VeraCrypt  
A disk encryption software that protects files and drives using strong encryption.

3. KeePass  
An offline password manager that securely stores user passwords locally.

4. ClamAV  
An antivirus tool that scans systems for malware.

5. Nmap  
A network scanning tool used to discover hosts and services on a network.

These tools are widely used by cybersecurity professionals for system analysis and protection.
