# A7 – Cryptography Used in Modern Networks

Modern networks use cryptographic protocols to protect communication.

Example observed: WiFi encryption (WPA2)

The home wireless network uses WPA2 encryption. WPA2 uses strong cryptographic algorithms to encrypt data transmitted between devices and the router.

This prevents attackers from intercepting or reading wireless traffic.

Network encryption is important because wireless communication can otherwise be easily intercepted by nearby attackers.
