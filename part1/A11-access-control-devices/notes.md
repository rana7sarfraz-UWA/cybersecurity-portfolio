# A11 – Access Control Devices

Access control devices restrict access to authorized individuals.

Five examples of access control devices are:

1. RFID Keycard Reader  
Used in offices and universities. Users tap a card to unlock the door.

2. Biometric Fingerprint Scanner  
Uses fingerprint recognition to verify identity before granting access.

3. Facial Recognition System  
Uses cameras and AI to identify authorized individuals.

4. PIN Code Keypad Lock  
Requires a numeric code to unlock doors.

5. Smart Lock  
A door lock that can be controlled using a smartphone application.

These devices improve security by ensuring that only authorized users can access protected areas.
