# A1 – Campus Security Concepts

Location: University campus

During observation around campus I discovered several security implementations.

1. CCTV Surveillance 
Security cameras are installed in hallways and entrances to monitor activity.

2. Keycard Access Control
Certain buildings require a student ID card to unlock doors. This prevents unauthorized people.

3. Secure Wi-Fi Authentication
The university Wi-Fi requires login details which ensures only authorized users can access the network.

4. Security Patrols
Campus security officers patrol the area to ensure safety.

These systems combine physical security and digital security to protect students, staff and infrastructure.
