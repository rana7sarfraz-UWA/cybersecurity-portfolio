# A14 – AI Enabled Security Solutions

Artificial intelligence is widely used in cybersecurity to detect threats automatically.

Examples include:

1. Darktrace – AI-based network threat detection.
2. CrowdStrike Falcon – AI-driven endpoint protection.
3. Microsoft Defender – uses machine learning to detect malware.
4. IBM QRadar – AI-powered security analytics platform.
5. SentinelOne – AI autonomous endpoint protection.

AI systems analyse large volumes of data to detect suspicious behaviour faster than manual analysis.
