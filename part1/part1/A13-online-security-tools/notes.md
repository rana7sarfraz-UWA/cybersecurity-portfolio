# A13 – Online Security Tools

Online security tools help protect users from cyber threats while browsing the internet.

Five examples include:

1. Cloudflare – protects websites using web application firewalls and DDoS protection.
2. VirusTotal – scans files and URLs for malware using multiple antivirus engines.
3. Google Safe Browsing – warns users about malicious websites.
4. Have I Been Pwned – checks if personal data has been leaked in breaches.
5. NordVPN – encrypts internet traffic and hides IP addresses.

These tools improve online safety by detecting threats and protecting user data.
