# A2 – Public Space Security

Location: Local shopping centre

Several security measures were observed in the public environment.

1. CCTV Cameras  
Security cameras monitor entrances, corridors and stores.

2. Security Guards  
Security personnel patrol the area to respond to theft or emergencies.

3. Alarm Systems  
Stores use alarm systems that trigger when unauthorized access occurs.

4. Electronic Access Gates  
Some areas use controlled access gates to regulate entry.

These security implementations protect public safety and reduce criminal activity.
