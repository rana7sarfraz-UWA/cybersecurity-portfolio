B23 — Intrusion Detection System Testing

A basic Intrusion Detection System (IDS) such as Snort was tested by generating simple network activity and observing whether alerts were triggered.
The system was used to monitor traffic and detect suspicious patterns such as scan-like behavior or unusual requests.
The IDS was able to log and flag certain activities, showing that it can identify potential malicious behaviour based on predefined rules.
However, normal traffic was not always distinguished from suspicious traffic without proper tuning.
Overall, the test showed that IDS tools are useful for monitoring and early detection, but their effectiveness depends on correct configuration and rule sets.
