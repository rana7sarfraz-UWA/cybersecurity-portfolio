B3 — Proactive Security Implementations

Three proactive cybersecurity implementations were identified and analysed based on their ability to prevent or detect attacks before significant damage occurs.
The examples included intrusion detection systems (IDS), multi-factor authentication (MFA), and Moving Target Defense (MTD).
These mechanisms improve security by increasing detection capability, strengthening authentication, and reducing predictability for attackers. 
Proactive security controls help minimise attack success and improve system resilience against evolving threats.
This activity demonstrated the importance of preventive and adaptive security measures in modern cybersecurity environments.
