B2 — Strong Security Implementations

Five strong security implementations were identified and analysed from modern systems and applications.
The examples included multi-factor authentication (MFA), password hashing, HTTPS encryption, access control lists (ACLs), and intrusion detection systems (IDS).
These implementations improve confidentiality, integrity, and authentication by reducing the risk of unauthorised access, credential compromise, and network-based attacks.
The activity demonstrated how layered security controls and secure configuration practices significantly strengthen system security.
