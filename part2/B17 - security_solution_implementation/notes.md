B17 — Security Solution Implementation

A modern cybersecurity solution was implemented by enabling Multi-Factor Authentication (MFA) on an online account to improve account security.
MFA adds an additional verification step beyond passwords, reducing the risk of unauthorised access from stolen or weak credentials.
The implementation involved configuring an authenticator application and linking it to the selected account. 
Login attempts were then tested using both password-only and MFA-protected authentication.
The implementation demonstrated how layered authentication significantly improves account security and reduces the effectiveness of credential theft attacks.
