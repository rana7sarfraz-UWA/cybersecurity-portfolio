B29 — CVE Fix Comparison Using AI Systems

A known CVE vulnerability was selected and analysed using multiple generative AI systems to compare the consistency of suggested fixes.
The same vulnerability description was provided to different AI tools, including ChatGPT and at least one other AI system, to evaluate differences in responses.
Each system provided mitigation steps such as patching, configuration changes, or disabling vulnerable features. The responses were compared based on accuracy, completeness, and consistency of the suggested fixes.
The results showed that while most systems provided similar high-level solutions, there were variations in depth and implementation detail, highlighting that AI-generated security advice may not always be consistent and should be verified using official sources.
