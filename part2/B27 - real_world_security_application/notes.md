B27 — Real-World Security Application

A cybersecurity concept learned in class, specifically access control and least privilege, was applied to a real-world environment by restricting access to sensitive files on a local system.
File permissions and user restrictions were configured to ensure only authorised users could access specific data.
This included setting stricter permissions on sensitive directories and testing access attempts from non-authorised users.
The results showed that restricted users were unable to access protected resources, confirming effective enforcement of security controls.
Overall, this demonstrated how theoretical security principles can be applied to practical system hardening scenarios.
